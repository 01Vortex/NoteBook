本仓库是一个系统化的编程学习笔记整理库，涵盖**前端开发、后端开发、数据库操作、服务器运维、Spring 全家桶技术栈、渗透测试实战以及完整项目开发流程**等多个方向。适用于**编程初学者、开发者、安全研究人员、运维工程师**等各类技术人员参考与学习。

> 📌 **目标**：打造一个全面、系统、持续更新的技术学习资源库，帮助大家提升技能、构建项目、应对面试与实战场景。

### 🔐 渗透测试
- 常见漏洞原理与复现（SQL注入、XSS、CSRF、文件上传、反序列化等）
- 安全工具使用指南（如 Burp Suite、Nmap、Metasploit、AWVS）
- CTF比赛技巧与靶场搭建（DVWA、WebGoat、vulhub）
- 内网渗透与提权实战（横向移动、域渗透、权限维持）
- 安全审计与防御策略（OWASP Top10、代码审计、日志分析）

### 🖥️ 前端
- HTML/CSS/JavaScript 核心知识总结与最佳实践
- Vue.js / React / Angular 框架深入实践与源码解析
- 响应式布局与组件封装技巧
- Web 性能优化（懒加载、CDN、Tree Shaking）
- 前端工程化（Webpack、Vite、ESLint、Prettier、TypeScript）

### 🚀 服务端
- Java / Python / Node.js / Go 等主流语言开发笔记
- RESTful API 设计规范与实现细节
- 接口安全与鉴权机制（JWT、OAuth2、Session/Cookie）
- 异步处理与消息队列（RabbitMQ、Kafka、Redis Pub/Sub）
- 微服务架构入门与 Spring Cloud 实践

### 🗃️ 数据库
- MySQL / PostgreSQL / MongoDB 使用与调优经验
- SQL语法与高级查询技巧（JOIN、子查询、窗口函数）
- 数据库事务、索引、锁机制与隔离级别
- ORM框架使用与性能优化（MyBatis、Hibernate、Sequelize）
- 数据备份、恢复与迁移策略（mysqldump、mongodump）

### 🌿 Spring 全家桶
- Spring Boot 快速开发与部署流程
- Spring Security 权限控制详解（RBAC模型、OAuth2集成）
- Spring Cloud 微服务架构实践（Eureka、Feign、Zuul、Config Server）
- Spring Data JPA / MyBatis 集成与多数据源配置
- Spring Boot 常用配置与性能调优（Tomcat连接池、日志管理）

### ⚙️ 服务器运维
- Linux 常用命令与系统管理技巧（CentOS/Ubuntu）
- Nginx / Apache / Tomcat 配置与性能优化
- Docker 容器化部署与编排（Docker Compose、Kubernetes）
- Shell 脚本自动化运维（定时任务、日志切割）
- 服务器安全加固与入侵检测（SSH加固、防火墙配置、fail2ban）

### 💼 项目开发
- 项目结构设计与模块划分（MVC、DDD、Clean Architecture）
- Git 使用规范与团队协作流程（分支策略、Code Review）

- 单元测试与接口测试（JUnit、Pytest、Postman、Swagger）
- CI/CD 自动化部署流程（GitHub Actions、GitLab CI、Jenkins）

## 📚 学习路线推荐（适合不同阶段）

### 👶 初级阶段
- 熟悉一门编程语言（Java/Python/JS）
- 学会使用 Git 进行版本控制
- 掌握基本的前后端交互逻辑（HTML/CSS + JS + HTTP请求）
- 搭建本地开发环境（IDEA/VS Code + JDK/Node + 数据库）

### 🧑‍💻 中级阶段
- 掌握至少一个后端框架（Spring Boot/Django/Express）
- 理解 RESTful API 设计与实现
- 学习数据库设计与查询优化
- 开发小型项目并部署上线（静态博客、留言板、个人网站）

### 🧠 高级阶段
- 掌握微服务架构与分布式系统设计（Spring Cloud、Dubbo）
- 熟悉 DevOps 流程（CI/CD、容器化部署、监控报警）
- 参与开源项目或公司级项目开发
- 掌握常见安全漏洞及防御方法（OWASP Top10）

### 🧪 实战阶段
- 搭建靶机进行渗透测试（vulhub、Hack The Box）
- 参加 CTF 比赛或漏洞挖掘项目
- 实现自动化脚本与工具开发（Python + Requests + Selenium）
- 构建完整的 Full Stack 项目（从需求到上线全流程）
